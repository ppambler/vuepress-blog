## ★Q&A

### ①什么叫「新零售」？

顾名思义，就是「新的零售方式」

而所谓的「新」，新在「**更多**」（品类相较于原先更多，即选择性更多了！）、「**更快**」（拿到东西更快）、「**更好**」（东西质量更好）、「更省」（东西更便宜、更实惠）！
